# 세션 A — 종가 기록 및 2주 검증 (평일 16:30 KST)

역할: 장 마감 후 추적 중인 모든 종목의 당일 시세를 기록하고, 2주 모의매매 검증을 판정한다.

## 절차

1. `prompts/common.md`를 먼저 읽는다.
2. `python3 scripts/pilot.py status` 로 추적 종목(tracked_codes) 목록을 확인한다.
3. 사용자 명령 처리: `python3 scripts/pilot.py process-control` 실행. 결과를 요약에 포함.
4. 추적 종목이 하나도 없으면 5~7단계는 건너뛴다.
5. 각 추적 종목의 당일 시가/고가/저가/종가를 구글 파이낸스에서 가져온다 (common.md 참고).
   코스피/코스닥 지수도 함께 확인해 휴장 여부를 판단한다. 휴장이면 기록하지 않고 종료.
6. 가져온 가격을 JSON 배열로 만들어 기록한다:
   ```
   echo '[{"code":"005930","name":"삼성전자","open":247000,"high":263500,"low":243000,"close":259000}]' | python3 scripts/pilot.py record-prices
   ```
7. 검증 판정: `python3 scripts/pilot.py evaluate`
   - `passed` 판정이 나오면 → 매매 추천 신호가 자동 생성된다. 요약 맨 앞에 굵게 알린다:
     "🎯 검증 통과: {종목명} — 2주 모의매매 수익률 {x}%. 매수 검토 추천"
   - `failed` 판정이 나오면 → 사유와 함께 간단히 보고한다.
8. 대시보드 갱신: `python3 scripts/pilot.py dashboard`
9. 커밋·푸시 후, 요약 작성:
   - 검증 통과/탈락 종목 (있으면 맨 앞에)
   - 진행 중 종목별 현재 수익률 한 줄씩
   - 처리한 사용자 명령

## 주의

- 이 세션은 분석·종목 선정을 하지 않는다. 기록과 판정만 한다.
- 신호가 생겼을 때 요약에 "매수하세요"가 아니라 "매수 검토 추천"으로 쓴다.
  최종 매매 판단과 주문은 사용자가 직접 한다.
