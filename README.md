# stock-pilot

한국 주식(코스피/코스닥) 자동 분석 · 2주 모의검증 · 매매추천 시스템.
Claude 예약 작업(자동 세션)이 평일마다 이 저장소를 읽고 쓰면서 작동한다.

## 파이프라인 (평일, KST)

| 시간 | 세션 | 역할 |
|------|------|------|
| 16:30 | A. 종가 기록·검증 | 추적 종목 시세 기록, 2주 모의매매 판정, 사용자 명령 처리 |
| 17:00 | B. 시장 분석 | 지수·뉴스로 시장 국면과 유망/회피 섹터 판단 |
| 17:30 | C. 회사 분석 | 유망 섹터에서 펀더멘털 기반 후보 발굴 |
| 18:00 | D. 차트 분석 | 후보 기술적 점검 → 통과 종목을 2주 모의매매에 편입 |

흐름: **B 시장 → C 회사 → D 차트 → (10거래일 모의매매) → A가 판정 → 통과 시 매수 추천 신호**

검증 규칙(config.json): 편입가 대비 **+7% 도달 시 조기 통과**, **-5% 터치 시 조기 탈락**,
10거래일 만료 시 **+3% 이상이면 통과**. 통과 종목만 매수 추천 신호가 된다.
실제 주문은 시스템이 하지 않으며, 사용자가 직접 한다.

## 모바일 대시보드

GitHub Pages: `https://art-web1104.github.io/stock-pilot/`
진행 중 검증, 수익률 추이, 매수 추천 신호를 확인하고
제외/승인/무시 명령을 내릴 수 있다 (다음 세션 실행 시 반영).

## 구조

- `prompts/` — 각 세션의 행동 지침 (여기를 고치면 세션 행동이 바뀐다)
- `scripts/pilot.py` — 상태 관리 엔진 (가격 기록, 손익 계산, 판정, 대시보드 빌드)
- `data/` — 분석 결과, 모의매매 기록, 가격 히스토리, 신호, 사용자 명령
- `docs/` — 대시보드 (GitHub Pages)

## 면책

이 시스템의 모든 분석과 신호는 정보 제공 목적이며 투자 권유가 아니다.
투자 손실의 책임은 투자자 본인에게 있다.
